# AGENTS.md

## Projeto
Aplicativo mobile de rastreamento e análise de hábitos em React Native.

## Stack assumida
- Expo
- React Native
- TypeScript
- AsyncStorage
- navegação simples por tabs

## Objetivo
Transformar os wireframes e docs em um MVP funcional, com foco em clareza de código, boa UX mobile e escalabilidade futura.

## Regras de implementação
- Priorizar simplicidade.
- Não instalar bibliotecas sem justificar.
- Reutilizar componentes antes de criar novos.
- Separar UI, lógica de negócio e persistência.
- Usar TypeScript com tipos explícitos para modelos principais.
- Manter nomes claros de arquivos, componentes e funções.
- Evitar abstrações desnecessárias no MVP.
- Pensar em futura migração para backend, mas sem implementá-lo agora.

## Estrutura esperada
- components para UI reutilizável
- services para persistência
- types para modelos
- utils para cálculos e helpers
- docs como fonte de verdade funcional

## Done when
- onboarding funcional
- criação de registro funcional
- estatísticas básicas funcionando
- configurações básicas funcionando
- build sem erro
- lint sem erro

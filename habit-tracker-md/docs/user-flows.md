# User Flows - Habit Tracker

## Fluxo 1 - Primeira configuração
1. Usuário abre o app pela primeira vez
2. Visualiza a proposta do rastreamento
3. Seleciona os hábitos que deseja acompanhar
4. Define o período de análise
5. Salva o rastreamento
6. É redirecionado para a tela principal

## Fluxo 2 - Registrar um hábito
1. Usuário entra na tela principal
2. Toca em "Novo registro"
3. Seleciona o hábito
4. Seleciona o mood
5. Seleciona a intensidade
6. Adiciona observação opcional
7. Salva
8. O registro aparece no histórico e impacta as estatísticas

## Fluxo 3 - Editar registro
1. Usuário abre um registro existente
2. Altera mood, intensidade ou observação
3. Salva
4. Estatísticas são recalculadas

## Fluxo 4 - Excluir registro
1. Usuário abre um registro existente
2. Toca em excluir
3. Confirma a exclusão
4. Registro é removido
5. Estatísticas são atualizadas

## Fluxo 5 - Ver estatísticas
1. Usuário entra na aba de estatísticas
2. Visualiza cards com indicadores
3. Filtra por hábito ou período futuramente
4. Observa padrões de frequência, mood e intensidade

## Fluxo 6 - Ajustar configurações
1. Usuário entra em configurações
2. Altera hábitos monitorados
3. Ajusta período de rastreamento
4. Define preferências básicas
5. Salva alterações
